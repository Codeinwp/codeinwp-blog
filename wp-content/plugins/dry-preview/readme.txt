== Description ==
Dry Preview is a simple plugin which boost up your productivity with just one click by enabling you to preview all of your changes in editor immediately.

==Documentation==
Documentation is located in directory docs in dry-preview directory.

= Basic Installation =
* Plugin folder in the WordPress plugins folder must be 'dry-preview.
* Upload folder 'dry-preview' to the '/wp-content/plugins/' directory
* Activate the plugin through the 'Plugins' menu in WordPress
* Set the plugin from 'Settings' => 'Dry Preview' menu in WordPress
