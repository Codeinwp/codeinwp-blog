Dry Preview on CodeCanyon, WordPress plugin:

Dry Preview Plugin


Installation instructions
--------------------------------------


Via WordPress Install:
-----------------------
1. In WordPress administration, open Plugins -> Add New
2. Switch from Search to Upload on the top of the page
3. Browse to the plugin package zip on your computer
4. Click Install Now button
5. Activate plugin Dry Preview from WordPress admin Plugins panel.


Via FTP:
-----------------------
1. Unpack this archive.
2. Upload dry-preview folder to your website wp-content/plugins folder.
3. Activate plugin Dry Preview from WordPress admin Plugins panel.
